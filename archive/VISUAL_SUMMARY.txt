┌──────────────────────────────────────────────────────────────────────────────┐
│                        TYPICAL SESSION FLOW                                  │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   1. Student Registration → Create STUDENT#<id>                             │
│      ↓                                                                       │
│   2. Condition Assignment → Random to 1 of 24 conditions                    │
│      ↓                                                                       │
│   3. Session Creation → Create SESSION#<id> with GSI keys                   │
│      ↓                                                                       │
│   4. Location Verification → GPS/QR/Manual check                            │
│      ↓                                                                       │
│   5. Baseline Assessment → Test prior knowledge (10 questions)              │
│      ↓                                                                       │
│   6. Content Delivery → 10-minute audio/text (4 segments × 2.5 min)        │
│      │                                                                       │
│      ├─ Intervention after Segment 1 (if 2.5min interval)                  │
│      ├─ Intervention after Segment 2 (if 2.5min or 5.0min interval)        │
│      ├─ Intervention after Segment 3 (if 2.5min interval)                  │
│      └─ Intervention after Segment 4 (always, all intervals)               │
│      ↓                                                                       │
│   7. Final Assessment → Same questions as baseline                          │
│      ↓                                                                       │
│   8. Post-Survey → Experience feedback                                      │
│      ↓                                                                       │
│   9. Calculate Learning Gain → Final - Baseline                             │
│      ↓                                                                       │
│  10. Update Statistics → Via DynamoDB Streams                               │
│                                                                              │
└──────────────────────────────────────────────────────────────────────────────┘

│                     KEY RESEARCH METRICS                                     │
├──────────────────────────────────────────────────────────────────────────────┤
│                                                                              │
│   Learning Outcomes:                                                         │
│   ├── Absolute Gain = Final Score - Baseline Score                          │
│   ├── Normalized Gain = (Final - Baseline) / (100 - Baseline)              │
│   └── Effect Size (Cohen's d) = (Mean₁ - Mean₂) / Pooled SD                │
│                                                                              │
│   Intervention Quality:                                                      │
│   ├── Dynamic: Question clarity, relevance, progression quality             │
│   └── Static: Answer depth, accuracy                                        │
│                                                                              │
│   Engagement:                                                                │
│   ├── Content completion rate (%)                                           │
│   ├── Pause/rewind counts                                                   │
│   └── Segment engagement scores (0-1)                                       │
